How to Use This App

Select Move Type: Choose the attacking type from the dropdown.

Search: Filter results by typing any part of a type name.

Search Features

  • Partial matches: “fly” matches Flying, Dragon/Flying, etc.
  • Multiple terms: “fire, steel” shows all Fire and all Steel matchups.
  • Negative terms: “!rock” hides all Rock matchups.
  • Dual types in any order: “fairy/flying” matches “Flying/Fairy”.
  • Spaces allowed: “dragon / steel, fairy / flying”.

All tables update instantly as you type.
